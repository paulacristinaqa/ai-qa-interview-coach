# MVP Scope

## Dentro do MVP

- Uso individual.
- Aplicacao web responsiva.
- PT-BR e ingles.
- Dashboard essencial.
- Simulador textual de entrevistas.
- Feedback estruturado.
- Guided Learning com dicas progressivas.
- Technical Lab inicial para API Testing, SQL e Test Design.
- Historico basico.
- Career Readiness Index inicial.
- Developer Diary basico.
- Exportacao Markdown.
- Execucao local documentada.

## Fora do MVP

- PT-PT e outros idiomas.
- Audio, pronuncia e webcam.
- Multiusuario ou SaaS.
- Integracao GitHub.
- RAG completo.
- Importacao de curriculo e vagas.
- Recomendacoes externas de cursos.
- Gamificacao avancada.
- Exportacao PDF ou DOCX completa.

## Hipoteses

- O primeiro usuario e uma candidata unica.
- O treino inicial sera textual para reduzir complexidade tecnica.
- A stack inicial sera confirmada por ADR antes de implementacao extensiva.
- O app deve nascer preparado para provedores de IA diferentes, mas sem acoplar o dominio a um fornecedor.
