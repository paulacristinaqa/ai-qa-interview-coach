# Definition of Done

Uma historia e considerada pronta quando:

- criterios de aceite foram atendidos;
- testes relevantes foram criados ou a ausencia deles foi justificada;
- documentacao afetada foi atualizada;
- dados sensiveis nao aparecem em logs, fixtures ou exemplos;
- erros esperados sao tratados de forma previsivel;
- fluxos principais foram verificados localmente;
- decisoes importantes foram registradas em ADR ou Developer Diary;
- o card Jira recebeu comentario com o que foi entregue.
